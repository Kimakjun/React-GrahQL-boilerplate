### React GraphQL BoilerPlate

### 개발 환경 설정
- client
    - React CRA typescript
    - eslint, prettier, 절대 경로 추가(craco)
    - graphql client 설정
    - graphql codegen 설정
- server
    - express apollo server
    - eslint, prettier, 절대 경로 설정
    - graphql schema
    - codegen 설정

### getting started

```
    cd client
    npm i
    npm run types
    npm run start
    
    cd server 
    npm i
    npm run dev
    
```
